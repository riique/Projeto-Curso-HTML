# UNES — Projeto de HTML

Um site institucional fictício de universidade construído apenas com HTML. O projeto demonstra como organizar conteúdo, navegação, imagens e formulário em múltiplas páginas sem depender de frameworks ou JavaScript.

## Páginas disponíveis

- **Home:** apresentação da universidade;
- **Quem somos:** objetivos, diferenciais e informações institucionais;
- **Contato:** formulário visual com e-mail, assunto e mensagem.

## Conceitos praticados

- estrutura básica de documentos HTML;
- links e navegação entre páginas;
- listas, imagens e formulários;
- organização de layout com tabelas, conforme a proposta didática do curso.

> [!NOTE]
> O layout com tabelas representa o exercício original. Em projetos atuais, CSS e elementos semânticos são as opções recomendadas para estruturar a página. O formulário também é apenas visual e não envia dados para um servidor.

## Como visualizar

```bash
git clone https://github.com/riique/Projeto-Curso-HTML.git
cd Projeto-Curso-HTML
```

Abra `index.html` no navegador. Não há dependências, servidor ou etapa de compilação.

## Estrutura

```text
├── index.html
├── quem_somos.html
├── contato.html
└── imagens/
```

## Contexto

Projeto educacional desenvolvido durante um curso de HTML na Udemy. A universidade, seus textos e dados são fictícios.
